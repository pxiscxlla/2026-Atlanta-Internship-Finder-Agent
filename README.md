# Atlanta Internship Finder Agent

The Atlanta Internship Finder Agent is an AI-powered career assistant designed to help undergraduate students find Summer 2026 internships within 50 miles of Atlanta, Georgia.

## What It Does
This agent helps students by:
- Collecting exactly 4 skill keywords
- Searching for internships posted within the last 3 weeks
- Filtering positions within 50 miles of Atlanta, Georgia
- Matching opportunities that require at least 2 of the student’s skills
- Presenting ranked internship listings
- Generating a tailored professional cover letter for each matching role

## Purpose
The purpose of this agent is to make the internship search process faster, more targeted, and more personalized for undergraduate students seeking relevant professional opportunities.

## Features
- Skill-based internship matching
- Location-based filtering
- Recent posting filtering
- Internship ranking by relevance
- Professional cover letter generation

## Target Users
Undergraduate students looking for Summer 2026 internships in the Atlanta area.

## Example Workflow
1. The student provides 4 specific skills
2. The agent searches for matching internship opportunities
3. The agent filters results by date, location, and skill match
4. The agent presents ranked job listings
5. The agent creates a custom cover letter for selected roles
